### Minimal typography blog theme
This is meant to be a minimalist blog theme

The `config.toml` should be setup as follows:
```
title = "title"
pygmentsCodeFences = true
pygmentsUseClasses = true

[params]
  description = "description"
  avatar = "img/url"
  favicon = "img/url"

[menu]
  [[menu.navigation]]
    name = 'link'
    url = 'link'
```
